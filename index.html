Input Method Entry (IME) Research

In the summer of 2017, I, under the guidance of Nikola Banovic, worked to create a model that estimates the speed of autocorrection algorithms on mobile phone keyboards. From this model, we wrote a paper that we submitted for peer review. It is an improvement over existing models because it more accurately considers the cost of making errors. This is more practical since no one can type perfectly all the time.

Based on this model, I am now working to create the optimal keyboard. Whereas other optimized IME's have been predicated on swapping the positions of certain keys, my work instead focuses on modifying the size of each individual key, thus maintaining the standard QWERTY layout of the keyboard. For a user, this approach is more natural since the standard QWERTY keyboard is so ubiquitous.

Engineers Without Borders (EWB) Drones

EWB is an on-campus organization at CMU which specializes in doing service projects that apply engineering concepts to solve real-world problems. The goal of this project is to enable a drone to fly over crop fields in Rwanda and to take pictures of them. The pictures will then be stitched together and then analyzed to help Rwandan farmers identify where they need to add more irrigation and where they can have less. To this end, I am working on developing and testing navigation software.


Baby Blue

This is a chess program that I wrote for my 15-112 Term Project. It has a fully functioning AI algorithm based on the classic Minimax Alphabeta pruning algorithm. The algorithm traverses all possible positions that can arise from a given position. From there, it evaluates each position and compares it to the best possible position that it had seen. The grader praised this project for having a "rock-solid UI".
