Projects

This is a selection of some of the projects that I am working on or have worked on and a brief description of what each does. Please click on each project to read a little bit more about it. More details can be provided upon request.

Input Method Entry (IME) Research

Building on some of my prior research, this project is looking to create the optimal QWERTY keyboard based on a model that we created over the summer. This research is an improvement over existing research because it does not transpose the positions of keys; rather, it leaves keys in the same place but manipulates the sizes of certain keys.

Behavior Dashboard

Built with Apache Spark and SQL, this project dynamically accesses a database of cancer data and displays them to clinicians to help them make better recommendations to patients. It enables the user to simulate "what-if" scenarios based on patients with similar histories.

Engineers Without Borders (EWB) Drones

EWB is an on-campus organization at CMU which specializes in doing service projects that apply engineering concepts to solve real-world problems. The goal of this project is to enable a drone to fly over crop fields in Rwanda and to take pictures of them. The pictures will then be stitched together and then analyzed to help Rwandan farmers identify where they need to add more irrigation and where they can have less.

Baby Blue

This is a chess program that I wrote for my 15-112 Term Project. It has a fully functioning AI algorithm based on the classic Minimax Alphabeta pruning algorithm. The algorithm traverses all possible positions that can arise from a given position. From there, it evaluates each position and compares it to the best possible position that it had seen.
